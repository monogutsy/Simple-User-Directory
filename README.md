# 👥 Simple-User-Directory

<p align="center">
  A simple React application that fetches and displays user data using the <b>JSONPlaceholder API</b>.
</p>

<p align="center">
  ⚛️ React • 🌐 Fetch API • 📋 User Directory
</p>

---

# ✨ Features

✅ Fetches user data from an API  
✅ Displays first 5 users  
✅ Shows user name, email, and company  
✅ Loading state handling  
✅ Error handling  
✅ Built with React Hooks

# 🚀 Getting Started

## 1️⃣ Clone the Repository

```bash
git clone https://github.com/your-username/Simple-User-Directory.git
```

---

## 2️⃣ Navigate to the Project Folder

```bash
cd Simple-User-Directory
```

---

## 3️⃣ Install Dependencies

```bash
npm install
```

---

## 4️⃣ Start the Development Server

```bash
npm start
```
